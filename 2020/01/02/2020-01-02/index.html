2019年终总结与展望

发表于 2020-01-02 | | 阅读次数:

2019年走了，2020年来了，时间总是过得特别快，不知不觉就又老了一岁。以前很少愿意去总结，也不喜欢立什么flag，因为自己是个很随意，又很懒的人。但眼看着要奔40去了，19年又是特别忙碌和有意义的一年，觉得还是该记录下什么，趁着陪孩子外出考试的机会，把19年的工作、学习和生活各个方面做个总结，对到来的2020年做个展望。

2019年度总结

工作方面

从研究所出来的这一年多工作下来，首先，感谢领导和同事的指导和信任，让我的工作有了明确的方向和动力；其次，感谢给力的团队，没有伙伴们的团结和努力，不会取得现在的成绩。

具体来讲，主要有几个方面：

  1. 探索行业知识图谱的构建和落地应用。图谱的构建和应用一定是数据和业务需求驱动的，幸运的是这些条件我们是具备的。这一年和工程经验丰富的同事一起调研图谱的底层架构选型、设计行业知识的本体结构、讨论图谱落地的应用需求，收获很大。同时，针对图谱构建和应用的各种算法问题，和团队的伙伴们分工开展攻关，取得了不错的进展，也进行了相应算法的技术储备。总体来看，我们开始逐渐形成行业大脑的整体雏形。
  2. 追踪NLP技术在工业界的前沿应用。2019年是NLP技术突飞猛进的一年，尤其是BERT为代表的预训练模型的普及，开始逐渐颠覆传统的算法应用模式。这一年我重点研究了预训练模型的相关技术，阅读了一些前沿的学术论文，在细粒度情感分析、文本语义匹配和文本生成的算法方面进行了一些行业实践，探索NLP技术在行业的创新应用。
  3. 参与成果的交流和分享。19年，参加了几次业界技术交流分享会，分别去上海、南京和苏州出差，对外分享团队的技术成果，扩大公司的影响力。年初和团队在国际语义评测比赛上也取得了一个子任务的第一名。对外的技术交流和分享给我最大的收获是能够从用户、商业、市场等各种维度去思考问题，而不是固守原有的技术思维，有的时候换个角度看事情，能得到完全不一样的想法，真是很有意思。

从结果看，虽然工作一直在推进，但落地应用上仍有欠缺，需要和产品同学更多碰撞交流。尽管业界前沿研究跟踪的比较紧密，但成果创新性上仍然不足。组织了几次组内交流，但频率较低，没有发挥大家的主观能动性。

学习方面

这一年学习上我觉得做的很不好，很多时间花在了微信、微博的碎片化阅读上面，看上去好像很多东西都知道点儿，但实际上都不精深，经不起检验。

具体来说，成果寥寥：

  1. 硬着头皮啃完了花书，但很多内容都联系不起来。
  2. 看了一些视频课，但没坚持。
  3. 论文看的很多，但没注意总结。
  4. 零散的看了几本电子书，包括时间管理、学习方法、理财、文学各方面，但有的还没读完。

生活方面

这一年生活方面的变动非常大，我最大的感受就是感恩，感谢老婆、长辈、儿子和新出生的小儿子。老婆这一年承受的精神和身体上的压力非常大，非常不容易，话不多说都在心里，除了感谢还是感谢。感谢父母、岳父对我们家庭的悉心照顾，让我们能够有更多时间投入到学习和工作中。感谢大儿子对我坏脾气的宽容，原谅我们对他的忽视和照顾不周。感谢新出生的小儿子，给我们生活增添了很多的乐趣。

这一年家庭的各种问题逐渐涌现出来，如何更好地和长辈相处、如何保持夫妻关系的和谐、如何教育子女这些问题以前思考的比较少，做的也非常不好，值得检讨学习。

2020年展望

2020年不想立什么具体目标，一切看行动，只是希望能在以下方面做的更好：

  1. 锻炼身体，保持好的精神状态，对家庭负责。
  2. 工作上脚踏实地，夯实基础，推进应用落地和成果创新。
  3. 学习上永远保持热情，不要懈怠，注重知识的体系化，提高动手实践能力。
  4. 生活上照顾好家人，凡事考虑周全，全心全意陪伴，能够为他们遮风避雨。
